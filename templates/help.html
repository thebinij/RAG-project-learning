Troubleshooting

Common Issues
  • ChromaDB Error: Run python init_vectordb.py
  • No Documents: Check legendary-docs/ folder
  • Port Conflict: Change port in app.py
  • Memory Issues: Reduce chunk size in settings
Debug Mode

Run with debug mode for detailed error messages:

export FLASK_ENV=development
python app.py
